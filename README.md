# wxChat_HarmonyOS

一. 介绍
  1. 项目使用 Harmony OS 5.0、 API 12 、ComponentV2。
  2. 项目使用DevEco Studio NEXT及其更高版本编译器。
  3. 仿微信，使用relationalStore数据库，存储微信用户信息，聊天信息，朋友圈信息。支持增删改查。
